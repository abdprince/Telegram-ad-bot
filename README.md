# Telegram Ad Bot

بوت تيلغرام لمنصة الإعلانات مع نظام عملات افتراضية

## المتطلبات
- Python 3.9+
- Railway.app account

## الإعداد
1. انسخ `.env.example` إلى `.env` واملأ المتغيرات
2. ثبت المتطلبات: `pip install -r requirements.txt`
3. شغل البوت: `python bot/main.py`
